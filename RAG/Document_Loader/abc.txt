To import TextLoader in LangChain for Python, use the following import statement:
Python

from langchain_community.document_loaders import TextLoader
This imports the TextLoader class from the langchain_community.document_loaders module. You can then instantiate it with the path to your text file and load the document content.
Example Usage:
Python

from langchain_community.document_loaders import TextLoader

# Create a sample text file for demonstration
with open("example.txt", "w") as f:
    f.write("This is a sample text file.\n")
    f.write("It contains multiple lines of text.")

# Instantiate the TextLoader with the file path
loader = TextLoader("example.txt")

# Load the documents
documents = loader.load()

# Print the loaded documents
for doc in documents:
    print(f"Source: {doc.metadata['source']}")
    print(f"Content: {doc.page_content}")
